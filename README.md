# Python Quiz Game

Command-line quiz game built with Python. The game asks random questions, tracks the score, and saves the last result using JSON.

## Features
- Multiple choice questions
- Randomized question order
- Score tracking
- Replay option
- Save and load last score (JSON)
- Menu system

## How to run
1. Install Python  
2. Run the file:
   `python quiz-app.py`

## What I learned
- Working with JSON (save/load data)
- Lists and dictionaries
- Loops and functions
- Random module (shuffle)
- User input validation
